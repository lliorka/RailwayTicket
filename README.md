## Описание задания

Разработать WPF-приложение для расчёта стоимости проезда в зависимости от:
- Расстояния (км)
- Количества билетов
- Типа комфортабельности (плацкарт / купе / полулюкс / люкс)

**Ставка проезда:** 8 рублей за 1 км  
**Коэффициенты комфортабельности:**
- Плацкарт — 1.0 (100%)
- Купе — 1.1 (+10%)
- Полулюкс — 1.2 (+20%)
- Люкс — 1.3 (+30%)

**Формула:** `Стоимость = Расстояние × 8 × КоличествоБилетов × Коэффициент`

## Запуск проекта

```bash
# Сборка решения
dotnet build RailwayTicket.sln

# Запуск тестов
dotnet test RailwayTicket.Tests/RailwayTicket.Tests.csproj

# Запуск приложения (требует Windows)
dotnet run --project RailwayTicket/RailwayTicket.csproj
```

## Автоматизированные тесты

Все тесты находятся в `RailwayTicket.Tests/TicketCalculatorTests.cs`.

Покрытые случаи:
- TC_CALC_1: Расчёт, плацкарт (коэффициент 1.0)
- TC_CALC_2: Расчёт, купе (коэффициент 1.1)
- TC_CALC_3: Расчёт, полулюкс (коэффициент 1.2)
- TC_CALC_4: Расчёт, люкс (коэффициент 1.3)
- TC_CALC_5–7: Несколько билетов
- TC_CALC_8: Граничное значение (минимум: 1 км, 1 билет)
- TC_CALC_9: Большие значения
- TC_CALC_10–13: Некорректные входные данные (исключения)
- TC_CALC_14: Проверка константы ставки
- TC_CALC_15: Параметризованный тест (6 наборов)
